**Ramznegar** is a monthly quantitative finance journal that analyzes major crypto and traditional assets using **hourly data**.

Each monthly issue is a **single Jupyter notebook** that contains all the code used to:

- Load data
- Compute metrics
- Generate plots
- Summarize results

---

## What Ramznegar Analyzes

For every month, Ramznegar studies:

**Assets**

- Bitcoin (BTC)
- Ethereum (ETH)
- Binance Coin (BNB)
- Gold
- Crude Oil
- S&P 500

**Using hourly data**, each notebook computes and visualizes:

- **Performance & Risk**
  - Returns (hourly, cumulative)
  - Volatility (realized, annualized)
  - Sharpe ratio

- **Tail Risk & Drawdowns**
  - Historical Value at Risk (VaR)
  - Maximum drawdown
  - Drawdown paths over the month

- **Distribution & Shape**
  - Empirical distribution of returns
  - Skewness & kurtosis
  - Comparison to a normal distribution

- **Regimes & Clusters**
  - Hidden Markov Models (HMM) for regime detection
  - K-means clustering on asset features

- **Cross-Asset Relationships**
  - Correlation matrices
  - Correlation heatmaps
  - Correlation and diversification analysis
